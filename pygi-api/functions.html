Lasem.cairo_set_source_pixbuf(cairo, pixbuf, pixbuf_x, pixbuf_y)¶
Parameters:
  • cairo (cairo.Context) – a cairo context
  • pixbuf (GdkPixbuf.Pixbuf) – a GdkPixbuf.Pixbuf
  • pixbuf_x (float) – X coordinate of location to place upper left corner of pixbuf
  • pixbuf_y (float) – Y coordinate of location to place upper left corner of pixbuf

Sets the given pixbuf as the source pattern for cairo.

The pattern has an extend mode of %CAIRO_EXTEND_NONE and is aligned so that the origin of pixbuf is pixbuf_x, pixbuf_y.

New in version 0.4.
